Currency Converter Web Application

Emanuel Valencia | Wdd330 BYU-Idaho

Purpose

The purpose of this web application is to allow users for the quickly conversion of any currency into any other currency in any given amount. Also, it is very useful to show travelers how their own currencies will fare when exchanged with other foreign currencies. Moreover, it can help businesses determine the selling and buying profits of different products.
